# Haul Alpha Reference Customer Feedback

Use this to create the required quantitative and qualitative feedback mechanisms for 10 reference customers.

## Quantitative Google Form

Create a Google Form titled:

```text
Haul Alpha Feedback
```

Use these questions:

1. **How likely are you to use Haul the next time you shop online?**  
   Linear scale: 1 = definitely not, 10 = definitely yes

2. **Which site did you test Haul on?**  
   Short answer

3. **Did the Save to Haul button appear on the product page?**  
   Multiple choice:
   - Yes
   - No
   - Only after refreshing
   - Not sure

4. **Did Haul capture the correct product name and price?**  
   Multiple choice:
   - Yes, both were correct
   - Product name was correct, price was wrong/missing
   - Price was correct, product name was wrong/missing
   - Neither was correct
   - I could not save the product

5. **What would make Haul worth paying $4.99/month for?**  
   Paragraph answer

Optional sixth question if your professor allows it:

6. **What confused you or broke while using Haul?**  
   Paragraph answer

## Qualitative Follow-Up Script

Ask 3 to 5 users these questions after they try the alpha:

1. Walk me through what happened when you installed Haul.
2. What shopping site did you try first, and why?
3. Did the Save to Haul button show up where you expected?
4. Did the side panel make sense?
5. Did the comparison dashboard help you decide anything?
6. What felt broken, confusing, or missing?
7. Would you use this again for a real purchase?
8. What would make you pay for Haul Pro?
