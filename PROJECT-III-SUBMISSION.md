## Canvas Text Entry

Our Project III Alpha Release for Haul is ready.

Haul is a Chrome extension that lets shoppers save products from different retail sites, view saved items in a side tray, and compare products side-by-side before making a purchase decision.

GitHub repository: https://github.com/ongcpatrick/haul

Alpha v0.1.0 release notes and install instructions: https://github.com/ongcpatrick/haul/releases/tag/v0.1.0

We have attached `haul-v0.1.0.zip`, which is the downloadable Alpha Release version 0.1 that can be installed locally in Chrome using Developer Mode and Load Unpacked.

Install instructions:
1. Download and unzip `haul-v0.1.0.zip`.
2. Open Chrome and go to `chrome://extensions`.
3. Enable Developer Mode.
4. Click Load Unpacked.
5. Select the `haul-extension` folder.
6. Visit a shopping/product page and click Save to Haul.

Release notes are available in the GitHub release and in `haul-extension/RELEASE-NOTES.md`.
